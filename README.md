# EcoPulse  
AI-Based Smart Energy Monitoring System  

## Theme  
Climate Tech Optimization Engines  

## Problem Statement  
Universities waste electricity due to lack of monitoring and prediction systems. Lights and ACs often run in empty rooms, leading to energy waste and higher carbon emissions.  

## Solution Overview  
EcoPulse uses smart meter data and AI models to:  
- Track energy usage  
- Predict peak usage hours  
- Detect unusual energy spikes  
- Provide optimization suggestions  

## System Architecture  
Smart Meter Data → Data Processing → AI Model → Anomaly Detection → Dashboard
